Phase 1 — every module to build (MVP)
Auth system
Google login, Facebook login, email/password. OTP SMS via Twilio. Passwordless for non-tech users. Role assigned at signup.
Phase 1Supabase Auth
Role-based access
4 roles: Owner, Instructor, Student, Parent. Each sees only their own screens. Spring Boot enforces this on every API call.
Phase 1
Instructor onboarding
Self-signup OR owner invites. School name, license number, SGI certification, vehicle info, working hours.
Phase 1
Student management
Add by instructor or owner. Name, contact, parent link, package purchased, lessons remaining, status.
Phase 1
Parent portal
Linked to student. Read-only: schedule, progress, payments. Gets WhatsApp alerts. No edit permissions.
Phase 1
Scheduling engine
Calendar view. Instructor sets availability. Student/owner/instructor can book, edit, cancel. Conflict detection auto-blocks double bookings.
Phase 1
WhatsApp reminders
One-tap sends pre-filled message to student + parent via WhatsApp link. Twilio SMS fallback. Auto-fires 24h before lesson.
Phase 1
Digital payment ledger
Track what's owed per session. Mark paid/unpaid. Cash + e-transfer support first. Stripe online payments later.
Phase 1
SGI evaluation engine
Pre-loaded SGI mistake categories. Big colour-coded buttons for in-car use. Auto-generates Road Test Readiness score.
Phase 1
Progress reports
Per-student PDF report card. Mistake history, improvement over time, readiness percentage. Shareable via WhatsApp.
Phase 1
Owner dashboard
Total revenue, unpaid sessions, active students, instructor workloads, monthly MRR summary.
Phase 1
Subscription billing
Free / $29 / $69 / $149 plans. Stripe billing. Plan limits enforced by Spring Boot. Upgrade prompts in-app.
Phase 1
Phase 2 — marketplace modules
First Car Matchmaker
Parent portal widget. Collects vehicle preferences + financing readiness. Pre-qualifies parents before student passes test.
Phase 2
Graduation trigger
When instructor marks "Passed Road Test" → auto-routes lead to partnered dealership CRM via API. $200–$500 bounty per conversion.
Phase 2
Instructor affiliate
Instructor earns $100 automated kickback when their student buys a car via SRoadTutor marketplace.
Phase 2
Dealership CRM API
REST API integration with dealership CRMs (DealerSocket, CDK). Lead payload: name, budget, location, timeline.
Phase 2
Tax prep integration
Export instructor earnings, session logs, payment records to CSV/API for accounting firms. CRA-ready format.
Phase 2
Insurance lead module
After passing, route student to partnered insurance brokers. Referral bounty per policy quote started.
Phase 2
Phase 3 — AI + data pipeline
New Driver Risk Score
Aggregate anonymized mistake logs across all students. Generate proprietary risk scores. License to auto insurers via B2B API. PIPEDA compliant.
Phase 3
AV edge-case telemetry
Sync timestamped mistake logs with Tesla/OEM vehicle telemetry (camera, steering angle). Sell labelled datasets to AV research firms.
